数据库学习之用户管理.md

mysql数据库学习之用户管理

创建用户

  • create user ‘用户名‘@’ip’ IDENTIFIED BY ‘密码’
  • grant select on 数据库名.表名 to ‘username‘@’登录主机’ identified by ‘密码’

以上两种方法都会向mysql.user表中插入数据,但是使用第二种方法可以同时进行权限赋予

修改密码

  • set password=password(‘新密码’)
  • update mysql.user set password=password(‘新密码’) where user=’用户名’ ,然后 flush privileges

第一种方法可以设置当前用户的密码

第二种方法可以设置多有用户的密码，多用于root用户操作

用户的权限管理

使用GRANT命令进行

  • grant 权限名称 on 数据库名.表名 to ‘username‘@’登录主机’ identified by ‘密码’
    权限名称可以是很多种：
    SELECT ,UPDATE,DELETE等
回收权限

关键字 REVOKE:语法同GRANT
